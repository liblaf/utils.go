    pkg install conda [flags]

    Options

      -b, --batch           run install in batch mode (without manual intervention), it is expected the license terms (if any) are agreed upon
  -f, --force           no error if install prefix already exists
  -h, --help            help for conda
  -p, --prefix string   install prefix, must not contain spaces. (default "/home/runner/.local/opt/conda")
  -s, --skip            skip running pre/post-link/install scripts
  -t, --tests           run package tests after installation (may install conda-build)
  -u, --update          update an existing installation
